# Credit Card Spending Analysis

## 📌 Objective
Analyze credit card transaction data to understand customer behavior, spending patterns, and key trends.

## 📊 Key Analysis
- Customer segmentation
- Spending trends by category
- Monthly transaction analysis
- High-value customer identification

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Jupyter Notebook

## 📁 Dataset
Dataset includes customer transactions, product categories, and spending amounts.

## 🚀 Results
- Identified top spending categories
- Found seasonal spending patterns
- Segmented customers based on transaction behavior

## 📌 Future Work
- Build predictive model for customer spending
- Credit risk analysis
